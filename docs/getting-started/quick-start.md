# Quick Start

Get SendStock AI running in 5 minutes.

## 1. Download & Install

Download the latest version for your platform from [sendstockai.com](https://www.sendstockai.com).

- **Windows** — `.exe` installer or [Microsoft Store](https://www.microsoft.com/store)
- **macOS** — `.dmg` installer

## 2. Create an Account

1. Open SendStock AI
2. Click **Sign In** and create a free account
3. Verify your email

## 3. Add Your First Files

Drag and drop your photos or videos onto the main window, or click **Browse** to select files.

Supported formats: `JPG`, `JPEG`, `TIFF`, `PNG`, `EPS`, `MP4`, `SVG`

## 4. Generate AI Metadata

1. Select one or more files
2. Click **Generate AI** (single file) or **Generate All** (batch)
3. Review the generated title and keywords
4. Edit if needed

::: tip
The AI generates metadata optimized for stock platforms. You can customize the style in **Settings → AI Prompt**.
:::

## 5. Export or Upload

- **Export CSV** — Download a CSV file ready for manual upload
- **Direct Upload** — Configure your FTP/SFTP credentials in **Settings → Platforms** and upload directly
