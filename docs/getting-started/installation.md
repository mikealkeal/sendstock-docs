# Installation

## System Requirements

| Platform | Requirement |
|---|---|
| Windows | Windows 10 or later (64-bit) |
| macOS | macOS 11 Big Sur or later |
| RAM | 4 GB minimum, 8 GB recommended |
| Storage | 500 MB free space |

## Windows

**Option 1 — Microsoft Store** (recommended)
1. Search for **SendStock AI** in the Microsoft Store
2. Click **Install**
3. Launch from the Start Menu

**Option 2 — Direct installer**
1. Download the `.exe` installer from [sendstockai.com](https://www.sendstockai.com)
2. Run the installer and follow the setup wizard
3. Launch **SendStock AI** from the Start Menu or desktop shortcut

## macOS

1. Download the `.dmg` file from [sendstockai.com](https://www.sendstockai.com)
2. Open the `.dmg` and drag SendStock AI to your **Applications** folder
3. On first launch, right-click the app and select **Open** (required on first run due to macOS Gatekeeper)

::: warning macOS Note
If you see a security warning on macOS, go to **System Settings → Privacy & Security** and click **Open Anyway**.
:::

## Updates

SendStock AI updates automatically. When a new version is available, you'll see a notification inside the app. Click **Install Update** to apply it.
