# Getting Started — Web App

The SendStock AI web app works directly in your browser. No installation required.

## Access

Go to [sendstockai.com](https://www.sendstockai.com) and sign in to your account.

## What You Can Do

- Generate AI titles, keywords, and descriptions for your images and videos
- Edit metadata manually
- Export metadata as CSV for manual upload to stock platforms

## Your First Generation

1. Drag and drop your files onto the main area, or click to browse
2. Click **Generate AI** on a file to generate its title and keywords
3. Review and edit the results if needed
4. Click **Export CSV** to download the metadata file

## Supported Formats

`JPG` `JPEG` `PNG` `EPS` `MP4` `MOV` `SVG`
