# Export CSV

Export your metadata as a CSV file for manual upload to stock platforms.

## When to Use CSV Export

Some contributors prefer to upload files via FTP and then submit metadata separately via CSV. SendStock AI generates a CSV file in the exact format required by each platform.

## How to Export

1. Load your files and generate metadata (titles + keywords)
2. Click the **Export CSV** button in the toolbar
3. Select the target platform format
4. Choose a save location
5. The CSV file is ready to upload in your contributor dashboard

## Supported Formats

| Platform | Format |
|---|---|
| Adobe Stock | Adobe Stock CSV format |
| Shutterstock | Shutterstock CSV format |

## CSV Contents

The exported file includes:
- Filename
- Title
- Keywords (comma-separated)
- Category (where applicable)

::: tip
Make sure filenames in the CSV match exactly the filenames of the uploaded files.
:::
