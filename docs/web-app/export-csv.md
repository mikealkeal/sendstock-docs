# Export CSV — Web App

Export your metadata as a CSV file ready to upload on stock platforms.

## How to Export

1. Load your files and generate metadata
2. Click the **Export CSV** button in the toolbar
3. Select the format matching your target platform
4. Save the file to your computer

## What's Included

The CSV contains for each file:

- Filename
- Title
- Keywords
- Description

## Using the CSV

Upload the file directly in your stock platform contributor dashboard alongside your media files.

::: tip Filename match
Make sure the filenames in the CSV match exactly the filenames you uploaded to the platform.
:::
