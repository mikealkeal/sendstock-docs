# AI Generation — Web App

Generate titles, keywords, and descriptions for your stock media files directly in the browser.

## What Gets Generated

- **Title** — an optimized title for the file
- **Keywords** — up to 50 relevant keywords
- **Description** — a descriptive text for the file

## How to Use

1. Load your files by dragging them onto the main area or clicking to browse
2. Click the **AI** button on a file card to generate all metadata at once
3. To generate only a specific field, use the individual buttons (title, keywords, or description)
4. Review and edit the results directly in the input fields
5. Click **Save** or proceed to export

## Batch Generation

Click **Generate All** in the toolbar to process all loaded files in sequence. You can monitor progress and stop at any time.
