# Installation — Desktop App

## System Requirements

| Platform | Requirement |
|---|---|
| Windows | Windows 10 or later (64-bit) |
| macOS | macOS 11 Big Sur or later |
| RAM | 4 GB minimum, 8 GB recommended |
| Storage | 500 MB free space |

## Windows

**Option 1 — Microsoft Store** (recommended)

1. Go to the [SendStock AI page on Microsoft Store](https://apps.microsoft.com/detail/9PM54KHLS7JF)
2. Click **Install**
3. Launch from the Start Menu

**Option 2 — Direct installer**

1. Download the `.exe` installer from [sendstockai.com](https://www.sendstockai.com)
2. Run the installer and follow the setup wizard
3. Launch **SendStock AI** from the Start Menu or desktop shortcut

## macOS

::: info Apple Silicon only
The macOS version requires an Apple Silicon Mac (M1 or later).
:::

1. Download the `.zip` from [sendstockai.com](https://www.sendstockai.com)
2. Extract the archive and move **SendStock AI** to your **Applications** folder
3. Launch SendStock AI from your Applications folder

## Updates

SendStock AI updates automatically. A notification appears inside the app when a new version is available — click **Install Update** to apply it.
