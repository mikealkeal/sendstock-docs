# Upscale — Desktop App

Increase the resolution of your images directly in the app before uploading to stock platforms.

## Why Upscale?

Stock platforms have minimum resolution requirements. Upscaling lets you submit files that would otherwise be rejected due to insufficient size.

## How to Use

1. Go to the **Upscale** tab
2. Drop your images or select them from the file list
3. Choose the upscale factor
4. Click **Upscale**
5. Save the output files

## Supported Formats

`JPG` `JPEG` `PNG`

## Notes

- Upscaling runs locally on your machine — no files are sent to any server
- A GPU significantly speeds up processing
- Output files are saved separately — original files are not modified
