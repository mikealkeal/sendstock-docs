# Background Removal — Desktop App

Remove the background from your images locally before uploading.

## How to Use

1. Go to the **Background Removal** tab
2. Drop your images or select them from the file list
3. Click **Remove Background**
4. Save the output files

## Supported Formats

`JPG` `JPEG` `PNG`

## Notes

- Background removal runs locally on your machine — no files are sent to any server
- Output is saved as PNG to preserve transparency
- Original files are not modified
- A GPU speeds up processing on large batches
